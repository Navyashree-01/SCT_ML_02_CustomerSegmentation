# SCT_ML_02_CustomerSegmentation
# Customer Segmentation using K-Means Clustering

## Objective
Group retail store customers based on purchasing behavior.

## Dataset
Mall Customer Dataset (Kaggle)

## Technologies
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Google Colab

## Algorithm
K-Means Clustering

## Features Used
- Annual Income
- Spending Score

## Outcome
Customers segmented into 5 groups based on spending patterns.
